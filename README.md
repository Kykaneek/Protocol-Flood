Ninniejsza aplikacja została stworzona na potrzeby przedmiotu **Bezpieczeństwo sieciowe** 

Składa się z 3 części: 

 - Aplikacji UI Windows Forms - instalujemy na stacji roboczej
 - Aplikacji TCPServer - instalujemy na serwerze
 - Flooder - instalujemy na serwerze

Każda z częśći aplikacji znajduje się na innym branchu, iż każda znich to jest osobny projekt <br>
Wszystkie aplikacje uruchamiamy w Visual Studio code W celu ich pierwornemu przetestowaniu
